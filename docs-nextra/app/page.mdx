## Get Started

<Steps>
### Clone the Repository
```bash
git clone https://github.com/identity-wael/neurascale.git
cd neurascale
```

### Set Up Environment
```bash
./scripts/dev-tools/setup-venvs.sh
```

### Start the Platform
```bash
docker-compose up -d
cd neural-engine && source venv/bin/activate
python -m src.main
```

### Connect Your Device
Visit `http://localhost:8000` to connect your BCI device and start streaming data.
</Steps>
